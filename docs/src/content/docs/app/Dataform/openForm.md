Opens the form UI for an existing dataform item.

### Parameters

| Parameters | Type   | Description                                                  |
| ---------- | ------ | ------------------------------------------------------------ |
| item       | Object | An object with `_id` (required) and an optional `_view_id`. |

### Syntax

```js
dataformInstance.openForm({ _id: "item_123" });
```
