Discards the current draft for the dataform.

### Parameters

| Parameters | Type   | Description                                    |
| ---------- | ------ | ---------------------------------------------- |
| viewId     | String | Optional. View ID for a view-specific discard. |

### Syntax

```js
await dataformInstance.discardItem();
```
