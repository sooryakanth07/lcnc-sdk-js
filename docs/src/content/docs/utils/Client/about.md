The `kf.client` interface provides methods to interact with the client, such as showing messages, confirmation dialogs, and redirecting the user.

The available methods are:

1. showInfo() — display a toast message.
2. showConfirm() — display a confirmation dialog.
3. redirect() — redirect the client to a URL.
4. getImageUrl() — resolve a Kissflow image field value to a base64 data URL.
