While integrating SDKs, the initial step involves defining the account and user details.

Use the utilities below to invoke the respective details and perform common operations:

1. [Account](/utils/account/) — retrieve the account ID.
2. [User](/utils/user/) — access the authenticated user's details.
3. [Roles](/utils/roles/) — list the app roles assigned to the user.
4. [API](/utils/api/) — make internal and external API calls.
5. [Client methods](/utils/client/about/) — show toasts, confirmation dialogs, and redirect the client.
